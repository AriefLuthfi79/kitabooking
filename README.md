## Overview
<p>Kitabooking is application for booking transportation system.</p>

## Installation
```sh
$ cp .env.example .env
$ composer install
$ setup your .env
$ php artisan migrate
$ php artisan migrate --path=database/migrations/new
$ php artisan key:generate
$ php artisan serve
$ run browser localhost:8000
```
